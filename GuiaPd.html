Estrutura de um Documento HTML

Um documento HTML básico é composto por várias tags que definem a estrutura e o conteúdo da página. Abaixo está um exemplo de um documento HTML simples:



!DOCTYPE html


html lang="pt-BR"


head

    
meta charset="UTF-8"

   
meta name="viewport" content="width=device-width, initial-scale=1.0"

   
title
Título da Página
/title


/head


body

   
h1
Título Principal

   
p>Este é um parágrafo.


/body


/html

Tags HTML Básicas

h1

A tag

h1

é usada para definir o título principal de uma página. É a tag de título de nível mais alto e deve ser usada para o título mais importante. Exemplo:

h1
Este é o Título Principal
/h1

p

A tag

p

é usada para definir um parágrafo de texto. Exemplo:

p
Este é um parágrafo.
/p

img

A tag

img

é usada para incorporar uma imagem na página. É uma tag auto-fechante e requer o atributo src para especificar o caminho da imagem e o atributo alt para fornecer um texto alternativo. Exemplo:

img src="caminho/para/imagem.jpg" alt="Descrição da Imagem" /

a

A tag

a

é usada para criar um link. O atributo href especifica o destino do link. Exemplo:

a href="https://www.example.com"
Visite o Example
/a

hr

A tag

hr

é usada para inserir uma linha horizontal, geralmente para separar conteúdo. É uma tag auto-fechante. Exemplo:

hr /

Como Usar Estas Tags

Estas tags são fundamentais para a criação de uma página HTML. Utilizando-as, você pode estruturar seu conteúdo de maneira organizada e semântica. Aqui está um exemplo de como elas podem ser usadas juntas em uma página:



!DOCTYPE html

&html lang="pt-BR"

head

    
meta charset="UTF-8"

   
meta name="viewport" content="width=device-width, initial-scale=1.0"

   
title
Exemplo de Uso das Tags
/title


/head


body

   
h1
Título Principal
/h1

   
p
Este é um parágrafo de introdução.
/p

   
img src="caminho/para/imagem.jpg" alt="Descrição da Imagem" /

   
p

/p
